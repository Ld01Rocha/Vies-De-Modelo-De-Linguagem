# Vies-De-Modelo-De-Linguagem
Este projeto utiliza modelos BERT (base e português) para identificar vieses algorítmicos em NLP. Através da técnica de mask filling, o sistema revela como a IA associa gêneros e etnias a estereótipos em tarefas de predição. É um estudo essencial sobre ética em IA, demonstrando como dados de treino podem replicar preconceitos sociais. 
OBS: Ele só não apresenta as saídas, porque o GitHub não suporta todos esses dados. E as pesquisas forem feitas para fins acadêmicos
